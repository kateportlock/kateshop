# Getting Started 

After you clone the project, please run: 
### `npm i` 
in the following directories:

* **root** (kateshop)
* **client**
* **server**

To run the project, open two terminals and simuntaniously run server and client:

Terminal 1:
### `cd server` 
### `nodemon server.js` 

Terminal 2:
### `cd client` 
### `npm start`
